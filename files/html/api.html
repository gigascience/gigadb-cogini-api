API

This document describes the REST API URLs that we implemented for GigaDB. The XML schema is available here.

Query the dataset by DOI

Please use the following URL syntax to query dataset by DOI,
http://gigadb.org/api/dataset/
For example, the URL http://gigadb.org/api/dataset/100039 will return a dataset associated with DOI 100039.

Query datasets by author name

Please use the following URL syntax to query dataset by author name,
http://gigadb.org/api/author/
For example, the URL http://gigadb.org/api/author/Luo, R will return all the datasets associated with author "Luo, R".

List all available datasets

Please use the following URL syntax to list all datasets available at GigaDB,
http://gigadb.org/api/dataset/

List all available sample information

Please use the following URL syntax to list all samples available at GigaDB,
http://gigadb.org/api/sample

Query datasets by sample name

Please use the following URL syntax to query dataset by sample name,
http://gigadb.org/api/sample/
For example, the URL http://gigadb.org/api/sample/Homo will return all the datasets associated with sample name like "Homo".

Query files by DOI

Please use the following URL syntax to query files by DOI,
http://gigadb.org/api/file/
For example, the URL http://gigadb.org/api/file/100039 will return all the datasets associated with author "Luo, R".
